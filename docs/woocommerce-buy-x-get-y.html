How to Set Up Buy X Get Y

The Buy X Get Y rule allows you to create cross-sell promotions where purchasing one product (Product X) grants a discount on a different product (Product Y).

1. Creating a New Rule

  1. Navigate to WooCommerce > Settings > Dynamic Discounts.
  2. Click Add New Rule.
  3. Enter a Rule Name (e.g., "Buy Laptop Get Mouse Free").

2. Configuring the Discount

In the Discount Type section, select:

Discount Type: Buy X Get Y (Different Product)

Setting Up Tiers (Ranges)

This section defines the "Buy" quantity and the "Get" reward.

  1. Click Add Range.
  2. Min Qty: Enter the quantity of Product X the user must buy (e.g., 1).
  3. Get Items:
    • Search and select the product(s) to be given as a reward (Product Y).
    • e.g., Search for "Wireless Mouse".
  4. Discount Type:
    • Free: Product Y is free.
    • Percentage: Product Y is X% off.
  5. Value: The percentage amount (if applicable) or ignored if "Free".
  6. Recursive: Check this if you want the deal to repeat (e.g., Buy 2 Laptops, Get 2 Mice).

3. Filtering Logic (Defining Product X)

Use the Filter section to define what "Product X" is.

  • Filter Type: Specific Products or Product Categories.
  • Operator: In List.
  • Selection: Select "Laptop" (or the product customer must buy at full price).

4. Common Examples

Scenario A: Buy Laptop, Get Mouse Free

  • Filter (Product X): Laptop.
  • Min Qty: 1.
  • Get Items (Product Y): Wireless Mouse.
  • Discount Type: Free.
  • Result: Customer adds Laptop to cart. Plugin automatically adds (or discounts if added) the Wireless Mouse for $0.

Scenario B: Buy Hoodie, Get Cap 50% Off

  • Filter (Product X): Hoodie.
  • Min Qty: 1.
  • Get Items (Product Y): Cap.
  • Discount Type: Percentage.
  • Value: 50.
  • Result: Customer buys a Hoodie, and gets the option to buy a Cap at half price.
Note: Depending on plugin settings, the "Get" item may need to be added to the cart by the customer manually, or the plugin may attempt to auto-add it. Check your General Settings.
