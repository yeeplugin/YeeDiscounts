How to Set Up Set / Bundle Discounts

The Set / Bundle Discount rule allows you to create special pricing when a customer buys a specific group of items together (e.g., "Buy Product A + Product B for $50").

1. Creating a New Rule

  1. Navigate to WooCommerce > Settings > Dynamic Discounts.
  2. Click Add New Rule.
  3. Enter a Rule Name (e.g., "Complete Outfit Bundle").

2. Configuring the Discount

In the Discount Type section, select:

Discount Type: Bundle (Set) Discount

Defining the Set

You need to specify which products make up the bundle.

  1. Use the Filter section to select the products.
  2. Typically, you select Specific Products and add the exact items (e.g., Shirt, Pants, Hat).
  3. In the Range or Condition area (depending on configuration):
    • Specify the quantity required for each item (usually 1 of each).

Setting the Bundle Price

Determine the reward for buying the set.

  • Discount Type:
    • Fixed Price: The entire set costs a specific amount (e.g., $99).
    • Percentage: Get X% off the entire set.
  • Value: The price or percentage amount.

3. Common Examples

Scenario A: Fixed Price Bundle

  • Items defined: Camera + Lens + Bag.
  • Discount Type: Fixed Price for Set.
  • Value: 500.
  • Result: Buying all 3 items costs $500 total, regardless of their individual prices.

Scenario B: Percentage Off Set

  • Items defined: Shampoo + Conditioner.
  • Discount Type: Percentage.
  • Value: 20.
  • Result: Buying both items typically grants 20% off both items.
Note: Check your cart visual. Some themes may display this as a line item discount or adjust individual item prices to match the total set price.
