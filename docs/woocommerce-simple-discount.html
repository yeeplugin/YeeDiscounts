How to Set Up a Simple Discount (Product Adjustment)

The Simple Discount (also known as Product Adjustment) is the most straightforward way to run specific sales. It allows you to apply a discount directly to products, categories, or your entire store.

Unlike other rules that might require minimum quantities, this rule changes the product price immediately for 1 item or more.

1. Creating a New Rule

  1. Navigate to WooCommerce > Settings > Dynamic Discounts.
  2. Click Add New Rule.
  3. Enter a Rule Name (e.g., "Summer Sale 20% Off").

2. Configuring the Discount

In the Discount Type section, select:

Discount Type: Product Adjustment

Adjustment Settings

Configure how the price changes:

  • Adjustment Type:
    • Percentage Discount: Reduces price by a percentage (e.g., 20% off).
    • Fixed Discount: Reduces price by a fixed amount (e.g., $5 off).
    • Fixed Price: Sets the price to a specific amount (e.g., everything is $10).
  • Value: Enter the numeric value for the discount.

3. Filtering Logic (Who gets the deal?)

This step is crucial. Use the Filter section to decide which products are affected.

  • All Products: Typically requires adding a filter for "Product Category" includes "All Categories" (or similar broad logic handled by plugin, depending on version).
  • Specific Category: Select Filter Type: Category, Operator: In list, and choose your category (e.g., "Hoodies").
  • Specific Products: Select Filter Type: Products, Operator: In list, and search for specific items.

4. Common Examples

Scenario A: Storewide 20% Off Sale

  • Discount Type: Product Adjustment
  • Adjustment Type: Percentage Discount
  • Value: 20
  • Filter: Category "In list" [Select all main categories].

Scenario B: $10 Discount on Hoodies

  • Discount Type: Product Adjustment
  • Adjustment Type: Fixed Discount
  • Value: 10
  • Filter: Category -> In List -> Hoodies.
  • Result: All hoodies are $10 cheaper. A $50 hoodie becomes $40.
Tip: Enable Show on-sale badge in Settings > Sale Badges to automatically display a "Sale!" badge on affected products.
