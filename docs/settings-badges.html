Sale Badges & Display Settings

Control how your discounts appear on the frontend. Go to WooCommerce > Settings > Dynamic Discounts > Settings and look for the Product Settings section.

1. Discount Pricing Display

Show discounted price (Strikeout)

This setting controls when the product price changes to show the "Strikeout" format (e.g., $50 $40).

  • Show only after a rule condition is matched exactly: Price only changes if the customer qualifies (e.g., has met minimum quantity).
  • Show on products covered... (Default): Indicates potential savings even before adding to cart (works best for Simple Discounts).

Show on pages

Select where the discounted prices should appear: Shop Page, Category Pages, or Single Product Page.

2. Sale Badges

On-sale badge trigger

Choose when to show the standard "Sale!" badge for these products.

Customize sale badge

Enable this checkbox to customize the badge text.

  • Display percentage: Check this to replace "Sale!" with the actual percentage (e.g., "-20%").
  • Enter sale badge text: Manually type custom text (e.g., "HOT DEAL").
  • Force override: Use this if your theme's default sale badge is overriding the plugin's badge.

3. Bulk Discount Table

If you use Bulk / Tiered Pricing, these settings control the quantity table.

  • Show bulk discount table: Enable/Disable the front-end table on product pages.
  • Position: Choose where to inject the table (e.g., Before Add to Cart form, After Meta, etc.).
