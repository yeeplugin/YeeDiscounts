General Settings

Configure global behaviors for YeeDiscounts rules. Go to WooCommerce > Settings > Dynamic Discounts > Settings and look for the General Settings section.

1. Discount Rule Logic

Option Name: Choose how the discount rules should apply

This setting controls what happens when a product matches multiple discount rules simultaneously.

  • Biggest one from matched rules (Default): The plugin checks all matching rules and applies ONLY the one providing the highest discount amount.
  • Lowest one from matched rules: Applies ONLY the rule providing the smallest discount.
  • First matched rules: Applies the first rule found in the execution order (based on Rule ID usually).
  • All matched rules: Applies ALL matching rules cumulatively.

2. Interaction with WooCommerce Coupons

Option Name: Choose discount rules behaviour with default WooCommerce Coupons

This determines if YeeDiscounts rules can coexist with standard WooCommerce coupon codes entered by the customer in the cart.

  • Run Both (Default): Both the dynamic discount and any valid coupon code will result in a discount.
  • Disable the coupons: If a dynamic discount rule is active/applied, standard coupon codes will be rejected or ignored.
  • Disable the rules: If a standard coupon code is applied in the cart, all dynamic discount rules will be removed/disabled for that order.
Note: If you are troubleshooting why a discount isn't appearing, check these settings first. Conflict resolution often happens here.
