Installation Guide

Thank you for choosing YeeDiscounts! Follow the steps below to install and activate the plugin on your WooCommerce store.

Requirements

  • WordPress: Version 5.8 or higher.
  • WooCommerce: Version 6.0 or higher.
  • PHP: Version 7.4 or higher recommended.

Option 1: Installation via WordPress Dashboard

  1. Log in to your WordPress Dashboard.
  2. Go to Plugins > Add New.
  3. Click Upload Plugin button at the top of the screen.
  4. Click Choose File and select the yeediscounts.zip file you downloaded.
  5. Click Install Now.
  6. After installation is complete, click Activate Plugin.

Option 2: Manual Installation via FTP

  1. Unzip the downloaded yeediscounts.zip file.
  2. Upload the yeediscounts folder to your server directory: /wp-content/plugins/.
  3. Log in to your WordPress Dashboard.
  4. Go to Plugins > Installed Plugins.
  5. Find "YeeDiscounts" in the list and click Activate.
Success! You should now see a new menu item WooCommerce > Dynamic Discounts or the YeeDiscounts menu depending on your version.
